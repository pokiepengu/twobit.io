Welcome to TwoBit

Twobit is a WIP Discord API, powered by a young programmer. This bot could get revolutionary changes as the programmer is still learning and could learn new things to implement into the API.

How to use TwoBit.

  1. First, you want to get the Discord Server you are sending it to ready. If you need the permissions to add the bot, ask a user who can grant certain permissions

  2. Once you are ready, go to this link https://discordapp.com/oauth2/authorize?client_id=677999681527218208&permissions=0&scope=bot.

  3. Select the server using the dropdown menu, and complete the verification.

  4. You now have TwoBit in your server.

  5. Use +help to view a list of commands to use.

If you cannot get the link to work, and you are sure that you cannot find another way. DM butterflux#1010 and I will supply you with a new link.
